## Getting Started

### Clone the repo

```bash
git clone https://github.com/MuhamedGamall/dribbble-clone
```

### Install dependencies

```bash
npm install
```

### Setup .env file

```env
# mongodb
DATABASE_URL=

# nextauth
NEXTAUTH_URL=
NEXTAUTH_SECRET=

# GOOGLE
GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=

# CLOUDUNARY
NEXT_PUBLIC_CLOUDINARY_CLOUD_NAME=
CLOUDINARY_API_KEY=
CLOUDINARY_API_SECRET=
CLOUDINARY_URL=
# app
NEXT_PUBLIC_APP_URL=

```

### Start the app

```bash
npm run dev
```
